Классическое машинное обучение (Classical ML)


Когда используется:

  • Есть данные
  • Данные простые
  • Признаки понятные

Обучение с учителем

Классификация

Популярные алгоритмы:

  • Naive Bayes classifier
  • Decision tree
  • Logistic regression
  • K-Nearest Neighbours
  • Support Vector Machine

Регрессия

Популярные алгоритмы:

  • Linear regression
  • Polynomial regression

Обучение без учителя

Кластеризация

Популярные алгоритмы:

  • K-means clustering
  • Mean shift
  • DBSCAN

Обобщение (Уменьшение Размерности)

Популярные алгоритмы:

  • Principal Component Analysis (PCA)
  • Singular Value Decomposition (SVD)
  • Latent Dirichlet allocation (LDA)
  • Latent Semantic Analysis (LSA, pLSA, GLSA)
  • t-SNE

Поиск правил (ассоциация)

Сегодня используют для:

  • Прогноз акций и распродаж
  • Анализ товаров, покупаемых вместе
  • Расстановка товаров на полках
  • Анализ паттернов поведения на веб-сайтах

Популярные алгоритмы:

  • Apriori
  • Euclat
  • FP-growth
